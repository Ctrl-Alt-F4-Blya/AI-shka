Подсказки для лучшего результата

1Обрежь лишнее вокруг текста.
2Для коротких кривых строк включай режим «Сложная строка».
3Для URL и e-mail выбирай режим «Ссылки и адреса».
4Для сложных фото включи «Локально + OpenAI API». Если API недоступен, программа оставит локальный результат.
