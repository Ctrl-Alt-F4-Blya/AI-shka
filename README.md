
AI-shka

AI-shka — локальный веб-сервис для распознавания текста с изображений, скриншотов и фотографий. Проект помогает извлекать текст, ссылки, цифры, короткие коды и отдельные строки из картинок с плохим качеством, размытием или сложным фоном.

Кратко о проекте

AI-shka запускается локально на компьютере и открывается как обычный сайт в браузере. Пользователь может загрузить изображение, вставить скриншот из буфера обмена или перетащить файл в рабочую область. После этого сервис обрабатывает картинку и выводит распознанный текст.

В проекте есть локальное распознавание, поддержка Tesseract/EasyOCR и возможность усилить результат через OpenAI API.

Возможности
загрузка изображения через файл;
вставка скриншота через Ctrl+V;
drag-and-drop загрузка;
распознавание русского и английского текста;
режимы для обычного текста, сложных строк, ссылок, номеров и коротких записей;
настройка масштаба, контраста, яркости, ч/б режима и инверсии;
вывод процента уверенности;
показ альтернативных вариантов распознавания;
поиск и отдельный вывод найденных ссылок;
копирование результата в буфер обмена;
озвучивание распознанного текста;
локальная работа через браузер;
возможность подключения OpenAI API;
скрипты для установки OCR-движков и сборки EXE.

Как запустить
Быстрый запуск на Windows
Скачайте или клонируйте репозиторий.
Откройте папку проекта.
Запустите файл:
START.cmd

Скрипт сам проверит Python, установит зависимости из requirements.txt и запустит локальный сайт.

После запуска откроется браузер. Если браузер не открылся автоматически, перейдите по адресу:

http://127.0.0.1:8765

Чтобы остановить сайт, закройте терминал.

Запуск вручную

Установите зависимости:

pip install -r requirements.txt

Запустите проект:

python main.py

Откройте в браузере:

http://127.0.0.1:8765
Дополнительная установка OCR

Для установки Tesseract OCR можно использовать:

INSTALL_OCR_ENGINE.cmd

Для установки более сильного локального распознавания через EasyOCR:

INSTALL_STRONG_OCR.cmd

После установки перезапустите START.cmd.

Настройка OpenAI API

Чтобы подключить распознавание через OpenAI:

Откройте файл OPENAI_API_KEY.txt.
Вставьте свой API-ключ вместо заглушки.
При необходимости настройте модель в OPENAI_API_SETTINGS.json.
Запустите проект через START.cmd.
На сайте выберите режим движка Локально + OpenAI API или OpenAI API с локальным fallback.

Важно: не загружайте настоящий API-ключ в публичный репозиторий.

AI-shka/
├── main.py                      # точка входа в приложение
├── requirements.txt             # основные зависимости Python
├── START.cmd                    # быстрый запуск проекта на Windows
├── INSTALL_OCR_ENGINE.cmd       # установка Tesseract OCR
├── INSTALL_STRONG_OCR.cmd       # установка EasyOCR и дополнительных инструментов
├── BUILD_EXE.cmd                # сборка проекта в EXE
├── OPENAI_API_KEY.txt           # файл для OpenAI API ключа
├── OPENAI_API_SETTINGS.json     # настройки OpenAI модели
├── static/
│   ├── index.html               # интерфейс сайта
│   ├── style.css                # стили интерфейса
│   └── app.js                   # логика загрузки, обработки и вывода результата
├── textvision/
│   ├── web.py                   # локальный HTTP-сервер и API
│   ├── ocr.py                   # основная логика распознавания
│   ├── image_tools.py           # обработка изображений
│   ├── local_model.py           # локальная модель распознавания символов
│   ├── easyocr_engine.py        # интеграция EasyOCR
│   ├── openai_vision.py         # интеграция OpenAI Vision
│   ├── samples.py               # работа с примерами
│   └── config.py                # настройки проекта
├── models/
│   └── glyph_bank.json.gz       # локальная база символов
├── samples/                     # тестовые изображения
└── tools/
    └── build_glyph_bank.py      # инструмент для сборки базы символов
Сборка EXE

Для сборки приложения в .exe запустите:

BUILD_EXE.cmd

После сборки готовый файл появится в папке:

dist/
